Divisibility Rule for 1

Divisibility Rule for 1: We generally perform divisibility test to find the factors of large numbers. Every number is not completely divided by other number it may leave fractional quotient. So, divisibility test for 1 statement is used to say that a number is divisible by 1 without performing the actual division operation. Here we are giving 1 divisibility rule definition with proof. Also, check the example questions on the divisivility rule for 1.

What is the Divisibility Rule for 1?

According to the divisibility rule of 1, every number is divisible by 1. This rule doesn't have any constraints. When a number is divided by 1 the answer is number itself, irrespective of the number size, type.

Example: If 28 is divided by 1, the result is 28 which is the original number.
