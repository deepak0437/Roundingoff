What is the Divisibility Rule for 22?

The divisibility test of 22 states that if the number is divisible by both 2 and 11 then it is divisible by 22. By using this divisibility rule for 22, you can easily determine the number is divisible by 22. Check the below steps and learn how to test divisibility by 22 along with a solved example.

Simple Steps on How Do You Check Number is Divisible by 22?

The following process will help you test whether the number is divisibility by 22 or not:

  • Firstly, the rule says the number must be divisible by 2 and 11.
  • Check for 2: If the last number is even like 0, 2, 4, 6, 8 then it is divisible by 2.
  • Check for 11: Do alternate subtracting and adding the digits in the number and the final result should be 0 or divisible by 11.
  • If both conditions are true then the number is divisible by 22. Look at the below module for a solved example and understand how the 22 divisibility test is applied.

Worked Out Example on How To Test If Number is Divisible by 22?

Example 1:

Find whether the 8-digit number 1862383A is divisible by 22 or not with the value of A.

Solution:

Given input

The 8-digit number is 1862383A.

Use the divisibility rule of 22 and check whether the number is divisible by 22 or not:

In the first step, check for the 2 divisibility rule:

According to the rule, the unit place digit should be an even number to be divisible by 2. Here A is the last digit so the possible values of A are 0, 2, 4, 6, and 8.

In the next step, check for 11 divisibility rule:

As per the rule of Divisibility by 11, we get

⇒ (8 + 2 + 8 + A) - (1 + 6 + 3 + 3) = 0 or multiple of 11

⇒ 18 + A - 13 = 0 or multiple of 11

⇒ 5 + A = 0 or multiple of 11

If A = 6 then the 8-digit number 1862383A is divisible by 22. Thus A value becomes 6.
