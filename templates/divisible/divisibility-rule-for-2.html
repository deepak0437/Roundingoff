Divisibility Rule For 2

What is the Divisibility Rule of 2?

The divisibility rule for 2 is a short form to check whether the number was divided by 2 without solving it in the division form. This rule says that the number that ends with 0,2,4,6,8 and so on with any prime number will be divisible by 2.

Proof of Divisibility Test of 2

We can prove this divisibility rule of 2 using an example.

Let us take any number as an example

Consider, the number 73882

According to the rule, if the last digit of the number ends with an even number then it is divisible by 2.

Now, as the given number ends with the even number, it is divisible by 2

73882 / 2 = 36941 with the remainder 0.

Since 73882 is divisible by 2.

Application of Divisible by 2 Rule

Let us look into some of the applications in that you can use the divisibility rule for 2.

Mary has 26 crackers that she wants to share with Nancy in break time. Is it possible for her and Nancy to split the crackers up evenly between them? If three of the crackers fall on the floor, and they need to be thrown out, is it possible to split up the remaining crackers evenly between Mary and Nancy?

Here we are going to split this application into two cases. In the first case, mary can split the crackers evenly with nancy as there are 26 crackers, as the divisibility rule applies. Because 26 ends with an even number. So they can evenly spread the crackers for both of them. In the second case as three crackers have fallen down, mary has only 21 crackers as the number does not end with an even number, they cannot be spread evenly for both of them.

FAQs on Test of Divisibility by 2

1. What is the rule for 2?

Any number that ends with the even numbers like 0,2,4,6,8… then the number will be divisible by 2.

2. Which of the following number is divisible by 2?

243, 456, 553, 772

456, and 772 are divisible by 2 as they end with multiples of 2.

3. How to explain the divisibility rule of 2 with an example?

As we know, the divisibility rule of 2 is the number which is having the end digit as an even number then it is divisible by 2.

Let us consider 345. As 345 ends with the odd number, it is not divisible by 2.
