What is the Divisibility Rule of 25?

The divisibility rule of 25 says if the number is divisible by 25 exactly when the last two digits formed are 0 or divisible by 25. Want to see the proof of the divisibility rule for 25? Check the below module.

Divisibility Test for 25 Proof

When the number is divisible by 25 then the last two digits should be divisible by 25 and the numbers are 00, 25, 50, or 75.

Proof:

If you consider 5150 as an example number then prove if it is divisible by 25 or not.

First, we assume this number to be written as

5150 = 5*1000 + 1*100 + 5*10 + 0

We can prove it depends on the below observation:

Remainder of 10i divided by 25 is 0 if ‘i’ greater than or equal to two.

Later state that 100, 1000,... etc lead to a remainder of 0 when divided by 25.

Thus, remainder of " 5*1000 + 1*100 + 5*10 + 0" divided by 25 is equivalent to remainder

of following :

0 + 0 + 50 + 0 = 50

Since 50 is divisible by 25, hence proved.
