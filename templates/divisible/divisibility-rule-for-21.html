What is the Divisibility Rule of 21?

There are two simple tricks to check whether the number is divisible by 21 or not. They are as follows:

Rule 1: If the number is divisible by both 3 and 7 then it is divisible by 21.

Rule 2: Multiply the last digit by 2 and subtract with the remaining number. You have to repeat this step till the result gets a single digit and the final number should be divisible by 21. Hence, the given number is divisible by 21.

How To Check If the Number is Divisible by 21 using Divisibility Test of 21?

The step-wise process of finding the number is divisible by 21 with the help of 21 divisibility rules is given below:

Rule 1:

  • As per the divisibility rule of 21, see the number is divisible by 3 by adding all the digits.
  • See the summation digit is divisible by 3, if it is divisible by 3 then it is divisible by 21.
  • Now, apply the divisibility rule of 7 ie., Given number excluding the unit digit - 2 × unit digit.
  • If the result is divisible by 7 then it is divisible by 21.
  • Remember when the number is divisible by 3 and 7 both then only it is divisible by 21.

Rule 2:

  • According to another trick of divisibility by 21, the last digit should be multiplied by 2.
  • Now, add the result to the rest of the given number.
  • If it is still a large number then repeat the above steps till it is divisible by 21.
  • When the result gets divisible by 21 then the original number is divisible by 21.

Solved Examples of 21 Divisibility Rule

Example 1:

Is 4598 divisible by 21?

Solution:

Given the number is 4598

Now, apply the divisibility rule of 21 ie., the number is divisible by both 3 and 7

First, check for divisibility by 3:

Sum of digits = 4+5+9+8 = 26

Verify if it is divisible by 3 ie., 26/3

26 is not divisible by 3

Secondly, check for divisibility by 7:

Double the last digit ie., 8+8 = 16

Subtract the result with the remaining digits in the number ie., 459 -16 = 443

Again double the last digit ie., 3+3 = 6

Subtract 44 - 6 = 38

Check if it is divisible by 7 or not i.e., 38/7

38 is not divisible by 7

Therefore, 4598 is not divisible by 21.

Example 2:

Is 525 divisible by 21?

Solution:

Given the number is 525

Let’s multiply the last digit by 2 ie., 5x2 = 10

Subtract 10 with the remaining number

Ie., 52 - 10 = 42

Check the above result is divisible by 21

Ie., 42 / 21

Yes, 42 is divisible by 21

Hence, 525 is divisible by 21.
