What is Meant by the Divisibility Rule of 23?

As per the divisibility rule of 23, multiply 7 with the last number and add the result to the remaining digits in the given number. Repeat the process if necessary and check the end result is divisible by 23. If yes, then the original number is also divisible by 23.

How to See if a Number is Divisible by 23 or Not?

If you are confused while checking if the number is divisible by 23 by applying the divisibility rule for 23. Check the step-wise process from below:

  • You can repeat the first step again and again if required.

Divisibility Test for 23 with Proof

Assume we have the input number as 345

Now, apply the divisibility rule of 23

In the next step, add the above result to the remaining digits
