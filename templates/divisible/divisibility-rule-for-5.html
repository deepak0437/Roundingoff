Divisibility For 5

Looking for help in finding whether the given numbers are divisible by 5 or not? Then have a look at our Divisibility Rule for 5. This test of divisibility by five statements is helpful to tell whether a number is completely divisible by 5 or not without performing the actual division process. Therefore, go through the divisivility rule of 5 definitions, how to find it, and worked out examples.

What is the Divisibility Rule for 5?

The divisibility of 5 states that if the units digit in the last or the last digit of the number is either 5 or 0, then that number is divisible by 5

Example: In 7895, the last digit is 5, so the number 7895 is divisible by 5. Similarly in 19680, the units digit is 0, hence the number is divisible by 5

Process to Test Divisibility by 5

The detailed step-by-step procedure to test the divisibility rule for 5 is listed here:

  • Let us take any number to test whether it is divisible by 5 or not.
  • Observe the last digit or digit at the unit place of the number.
  • If it is either 0 or 5, the number is divisible by 5.
  • Otherwise, the given number is not divisible by five.

Worked Examples on Test of Divisibility by 5

Example 1:

Check whether 725 is divisible by 5 or not.

Solution:

Given number is 725

As per the divisibility rule of 5, the unit's digit of the number should be either 0 or 5.

The last digit of 725 is 5

So, 725 is divisible by 5.

Example 2:

Check the divisibility of the following numbers by 5.

  1. i) 583
  2. ii) 1640

iii) 2794

Solution:

Using the divisibility test for 5, the last digit of the number should be either 5 or 0 to satisfy the divisibility rule.

  1. i) The last digit of 583 is 3 which is neither 5 nor 0, hence 583 is not divisible by 5.
  2. ii) The end digit of 1640 is 0 which satisfies the 5 divisibility rule.

Therefore, 1640 is divisible by 5.

iii) The digit at the units place is 4 which is neither 0 nor 5

So, 2794 is not divisible by 5.

FAQs on Rules on Divisibility by 5

1. What is the divisibility rule of 5?

If the number ends in a 0 or 5, then the number is divisible by 5.

2. How do we know if a number is divisible?

A number is divisible by another number means the number can be divided equally by that number. Or the result after division should be a whole number.

3. How to know if a big number is divisible by 5?

Irrespective of the number size, you can apply the divisibility test for the 5 rule. If the number ends with 0 or 5, then it should be divisible by 5. So, only observe the unit digit of the number.
