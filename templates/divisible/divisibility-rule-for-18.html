What is the Divisibility Rule for 18?

According to the divisivility rule of 18, a number is exactly divisible by 18 if it is divisible by both 2 and 9. In other words, we can put it as a number is divisible by 18 if it is an even number and the sum of digits is divisible by 9. Both these conditions have to be satisfied for a number to be divisible by 18.

How to Test Divisibility Rule of 18?

In order to check if a number is divisible by 18 or not, you need to keep these tips in mind and they are listed below

  • Firstly, before checking if a number is divisible by 18 see if they are divisible by both 2 and 9.
  • In order to be divisible by 2 the last digit of the number must be even.
  • In the same way, for a number to be divisible by 9, the sum of the digits must be divisible by 9.
  • If both the conditions are satisfied then a number is divisible by 18.
