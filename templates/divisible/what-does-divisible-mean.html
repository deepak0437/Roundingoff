What is meant by Divisible or Divisibility?

A Number is said to be divisible by another number if it leaves a remainder of 0. We have a certain set of rules namely Divisibility Rules that tell if a particular number is divisible by another number or not.

List of Divisibility Rules for Numbers 2 to 10

Below we have mentioned some of the divisibility rules for various numbers starting from 2 to 10. Go through them and check if a number is divisible or not easily next time even in case of large numbers.

  • Divisible by 2: If the number is even or ends in digits 0, 2, 4, 6, or 8 it is divisible by 2.
  • Divisible by 3: If the sum of all the digits in the number is divisible by 3 then the number is divisible by 3.
  • Divisible by 4: If the last two digits of the number are divisible by 4 then the number is divisible by 4.
  • Divisible by 5: If the last digit of the given number is either 0 or 5 then the given number is divisible by 5.
  • Divisible by 6: If the given number is divisible by both 2 and 3 then it is divisible by 6.
  • Divisible by 7: If the difference between the last digit doubled and the rest of the digits is divisible by 7 then the number is divisible by 7.
  • Divisible by 8: If the last three digits of the given number are divisible by 8 then the given number is divisible by 8.
  • Divisible by 9: If the sum of the digits of the given number is divisible by 9 then the given number is divisible by 9.
  • Divisible by 10: If the last digit of the given number is 0 then it is divisible by 10.

Below is the list of fun facts that you need to know regarding Divisible Rules. They are as follows
