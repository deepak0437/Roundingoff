Examples on Number to Scientific Notation

A large number can be written as follows:

ln = c x bn

ln = Large number

c = Coefficient

b = 10

n = exponent

The large number is reduced to a fractional number multiplied by number 10 with a postitive or negative exponent.

Here is an example to illustrate:

1,234,000,000.000

= 1.234 x 1012

= 1.234e+12

Scientific Notation to Number

To change a number from scientific notation to standard form, move the decimal point to the left (if the exponent of ten is a negative number), or to the right (if the exponent is positive). You should move the point as many times as the exponent indicates.
