What is Sequential Percentage? 

The Sequential percentage tells you how to increase or decrease a number by a percent, and then consider the result of that and increase or decrease by another percent. Here, you will observe that two or more percentage changes are applied to a quantity consecutively. 

Sequential Percentage Formula 

The sequential percentage Formula is entirely based on which type of change is taking place in the given numbers. Mainly we focus on

Percentage change = [(change in value)/original value] × 100%

Where Change in value = Final value – original value

OR 

When there is a percentage increase in the value then we use, 

Percentage Increase = [(Final value – Original value)/Original value] × 100%

When there is a percentage decrease in the value then we use: 

Percentage decrease = [(Original value – Final value)/Original value] × 100%

Illustrated examples For Sequential Percentage Calculator

Example: 

The number increased by 20% and then decreased by 20%. What will be the net increase and increase? 

Solution: 

Using the formula, 

Net increase or decrease percent = {(Original number or Final number) - (Final number or

original number) ÷ Original number} × 100

Let the original number be 100

Given: number is increased by 20%

Thus, New Number = 100 + 20% of 100

⇒ 100 + 20

⇒ 120

Given: number is decreased by 20%

Final number = New number – 20% of new number

⇒ 120 – 20% of 120

⇒ 120 – 24

⇒ 96

∴ Net Decrease in percentage = { (100 – 96) / 100} × 100

⇒ {4/100} × 100

⇒ 4%

Hence, The net decrease percentage is 4% which can be termed as -4%. 

Example: 

Jenny’s salary was increased by 10% and then decreased by 10%. What will be the net increase or decrease percent? 

Solution: 

Let Jenny’s original salary be 100. 

Given: Decreases by 10% and Increased by 10%. 

Let the number be 100

Increase in the number =10%

=> 10% of 100=10

Thus, Increased number = 100 + 10 = 110

This number is then decreased by 10%

Thus, Decrease in the number

=> 10% = 10% of 110 = (10/100×100) = 11

∴ New number = 110 - 11 = 99

Thus, the net decrease = 100 - 99 = 1

Hence, the net percentage decrease

=> ( 1/ 100 x 100 )% = 1% or -1% ( -ve used for decrease)

1. How sequence percentage can be calculated?

2. What is meant by sequential percentage? 

The sequential percentage can be termed as when the consecutive quantity has been affected by two or more percentage changes. Thus, you may see that the change is not just an addition of two percentage changes. This is because the base changes after the first phenomenon are applied over the quantity. 

3. What are the types of sequential percentage changes? 

Mainly there are three types of percentage changes that occur. These can be termed as: 

  • When both percentage changes in the quantity are positive. 
  • When both percentages change the quantity is negative. 
  • When one percentage changes in the quantity are positive and the other one is negative.
