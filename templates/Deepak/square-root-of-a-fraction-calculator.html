What is the Square Root of a Fraction? 

The square root of a fraction can be calculated by finding the square of the numerator and denominator separately and then dividing them with each other. To find the square root of the fraction you can use the below-listed equation: 

√fraction = √numerator / √denominator

How to Find the Square Root of a Fraction manually? 

Manual Calculations to find the square root of the fraction is quite easy. You can refer to the below steps to find the square root of fractions. 

  • First, you need to convert the given numbers into a separate form.

=> √(a/b)  =  √a / √b

  • Now, find the perfect squares of the numerator and denominator. 

=> a = z x z 

=> b = m x m 

  • Then, divide the numbers to get the answer.

NOTE: The number comes out from the root in the single form. 

Illustrated Example To Find Square Root of a Fraction 

Example: 

Find the square root of 4/9. 

Solution: 

Given: Fraction = 4/9

To find the √(4/9) you can refer to the below formula: 

=> √fraction = √numerator / √denominator

=> √(4/9)  =  √4 / √9

After finding the prime factors of 4 and 9 we get

=> 4  =  22

=> 9  =  32

Then, we have, 

=> √(4/9)  =  √22 / √32

=> √(4/9)  =  2/3

Thus, the square root of 4/9 is 2/3.

To find the square root of √(4/16) you can refer to the below steps: 

=> √(4/16) = √4/√16.

=> √(4/16) = 2 /4

Thus, by solving 2/4 we will get ½, which is equal to 0.5. 

Hence, the square root of √(4/16) is 0.5
