How to find the Square Root of a Number Squared?

Calculating the square root of a number that is squared is extremely easy because the square root is simply the inverse of a square. So, when you find the square root of a squared number, both these operations become nullified, revealing only the number inside as the final answer. 

  • We know that squaring a number is raising it to the power of 2. Square of x = x^2
  • Then, we know that we must raise the number to the power of ½ when we find the square root. So, the equation becomes like this: Square root of squared number = (x^2)^½ 
  • According to the rules of exponents, we can multiply both powers together. 

            Square root of squared number = x^(2× ½) = x.

So, the final formula is as follows: Square root of squared number = x^1

            Or √(x^2) = x 

In layman’s terms, the square root of any squared number is equal to the number itself. There is no extra calculation involved, so it is very simple. All you need to do is plug the number of your choice in as x, and you will get your answer.
