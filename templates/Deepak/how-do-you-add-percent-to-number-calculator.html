Illustrated Example on Adding Percent To Number Calculations

Example 1: 

You bought an item for $10 and resale it with a 10% profit. Find the selling price using the plus percent formula.

Solution: 

Add a Percent to the Number Calculator helps you find the selling price of 10 dollars at 10 percent easily. If you want to learn the calculation process check here: 

Given amount = 10 dollars

Percent = 10%

The formula to add a percent to a number is 

Total = x + (( y / 100) * x ) 

Now, substitute the given values in the above equation;

Selling price = 10 + [ (10 / 100) * 10 ]

          = 10 + 1

          = $11

Hence, the selling price is equal to 11 dollars.

How to Add 3 percent to the 200 number manually? 

Solution: 

Given data

Number = 200

Percent = 3%

Convert percent to decimal, and we get

3/100 = 0.03

Now, add one to the decimal number ie., 1+ 0.03 = 1.03

To find the plus percent value, multiply the number with the above result.

We get; 200 x 1.03 = 103.

Therefore, 200 plus 3 percent is 206.
