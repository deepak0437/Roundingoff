How to Round off to the Nearest Mile?

Follow below instructions to round off the nearest mile easily and quickly without any confusion.

  • Check for the decimal number that was given in the question.
  • Then, look for the units place before the decimal point.
  • After the decimal point, if the tents place value is >=5 then we need to round up the value by adding +1 and remove all the digits.
  • If the tenths place value is <=4, simply remove all the digits after the decimal point.

Solved Example for Rounding off to the Nearest Mile

Example

Question : Round off 55.78 to the nearest mile?

Solution:

Given, input values are 55.78

Here in tenths place we have 7 which is >5. So, add the value by one and remove all the digits after the decimal point.

Thus, 55.78 rounded to the nearest mile is 56
